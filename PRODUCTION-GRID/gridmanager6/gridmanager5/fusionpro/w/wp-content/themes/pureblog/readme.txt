== Description ==

Introducing PureBlog for bloggers wanting a theme for blogging that is simple, yet filled with an extensive array of features. Originally built from the popular Twenty Seventeen theme, PureBlog extends the theme to give you more flexibility.


== Screenshots ==
1. Main screenshot for the theme on the theme list section of the admin


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
